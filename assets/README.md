# Assets Folder

Tempatkan file-file berikut di folder ini:

## Profile Photo (Required for Hero Section)
- **Filename**: `profile.jpg` atau `profile.png`
- **Description**: Foto profil profesional Anda untuk Hero Section
- **Recommended Size**: 400x400px atau 500x500px (square/persegi)
- **Format**: JPG atau PNG
- **Quality**: High resolution, professional photo
- **Tips**:
  - Gunakan foto dengan background solid atau blur
  - Pastikan wajah terlihat jelas dan centered
  - Gunakan lighting yang baik
  - Dress code profesional (kemeja/blazer)
  - Ekspresi friendly tapi profesional
  - **Jika tidak ada foto**: Placeholder dengan inisial "MFA" akan muncul otomatis

## Tips:
1. Gunakan nama file yang deskriptif
2. Optimize ukuran file untuk loading cepat
3. Gunakan format yang sesuai (PDF untuk CV, JPG/PNG untuk gambar)
4. Pastikan file tidak terlalu besar (max 5MB untuk CV, max 500KB untuk images)
